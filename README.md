# Zerdalitepe Aile Sağlığı Merkezi

Mamak ilçesi Zerdalitepe bölgesinde hizmet veren Aile Sağlığı Merkezi'nin resmi web sitesi.

## Hakkında

Statik HTML/CSS/JS ile geliştirilmiş, tarayıcıdan doğrudan açılabilen veya herhangi bir statik hosting (GitHub Pages, Vercel, Netlify) üzerinden yayınlanabilen kurumsal bir tanıtım sitesidir.

## İçerik

- Ana Sayfa (slider + hizmetler + kadromuz)
- Kurumsal (Hakkımızda, Hizmet Standartları, Öncelikli Hastalar)
- Kadromuz (Aile Hekimleri, Aile Sağlığı Çalışanları, Yardımcı Personel)
- Dosyalar (ASM Talimatları, İlk Yardım, Sigaranın Zararları)
- Sağlık Üzerine (Aşı Takvimi, BKİ Hesaplama, Gebelik, Yenidoğan vb.)
- Duyurular, Galeri, İletişim

## Kullanım

Yerel önizleme için:

```bash
cd zerdalitepe-asm
python3 -m http.server 8000
```

Tarayıcıdan `http://localhost:8000` adresini açın.
